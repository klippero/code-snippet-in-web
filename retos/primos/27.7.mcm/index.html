El mínimo común múltiplo de una lista de enteros se determina calculando los factores primos de todos los enteros y tomando los factores comunes y no comunes con el mayor exponente.

A continuación se muestran varios ejemplos:
12 = 22 x 31
8 = 23
mcm = 23 x 31 = 24

10 = 21 x 51
27 = 33
32 = 25
mcm = 25 x 51 x 33 = 4320

3 = 31
7 = 71
9 = 32
mcm = 32 x 71 = 63

Implementa el método mcm en la clase ListaEnteros que obtenga los factores primos de todos los enteros de la lista y tome todos con el mayor exponente para calcular el mínimo común múltiplo.

Para implementar el método mcm, el proceso debe ser:

  1. inicializamos un diccionario vacío donde se irá construyendo el resultado con todos los factores comunes con el mayor exponente
  2. para cada número de la lista:
    1. se calculan los factores primos
    2. para cada factor primo
      1. si no está en el diccionario resultado, lo metemos con su exponente
      2. si está en el diccionario, nos quedamos con el exponente más alto
  3. desfactorizamos el diccionario para calcular el resultado que devolvemos
